# 📂 Project Data

This directory contains the dataset used for the **Smart Coffee Shop** analysis project.

## ☕ Dataset Source: Coffee Shop Sample Data
The raw data files in this folder were sourced from Kaggle for educational purposes.

* **Primary Source:** [Kaggle - Coffee Shop Sample Data](https://www.kaggle.com/datasets/ylchang/coffee-shop-sample-data-1113)
* **Scope:** 9 CSV files covering transactions, pastry inventory, customer loyalty, and sales targets for April 2019.
* **Usage:** This data forms the basis for the relational database schema in the `SmartCoffeeShop` MariaDB instance.

---
*Note: This is synthetic data provided for practice in data cleaning, SQL querying, and retail analytics.*
