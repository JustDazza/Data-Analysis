## 🔍 1. Understanding Data vs. Information
As per NCFE standards, raw data is the building block, while information is the result of processing.

* **Data**: Raw, unorganised facts (e.g., the value `452.7`).
* **Information**: Data that has been processed and given context (e.g., `Coffee Beans Inventory` weighs `452.7g`).

## 📊 2. Qualitative vs. Quantitative Data
Understanding the difference between numerical and descriptive data is essential for effective analysis.

### **Quantitative Data (Numerical)**
Data that can be measured or counted.
* **Discrete**: Whole numbers (e.g., **42** Flat Whites sold today).
* **Continuous**: Precise measurements (e.g., Milk temperature at **65.5°C**).

### **Qualitative Data (Descriptive)**
Descriptive data that provides "quality" insights.
* **Examples**: Roast levels (Light, Medium, Dark) or customer feedback comments.

## 📈 3. The Four Types of Data Analytics
Businesses use different analytics types to drive decision-making.

| Type | Question | Business Application |
| :--- | :--- | :--- |
| **Descriptive** | What happened? | Calculating total daily revenue. |
| **Diagnostic** | Why did it happen? | Investigating why sales dropped during a specific hour. |
| **Predictive** | What will happen? | Forecasting next week's milk order based on trends. |
| **Prescriptive**| How can we make it happen? | Suggesting a promotion to increase afternoon foot traffic. |

## 📏 4. Measuring Data & Big Data
* **Units of Measurement**: Data is measured in bytes (KB, MB, GB, TB).
* **Big Data (The 5 Vs)**: Understanding large-scale datasets through **Volume, Velocity, Variety, Veracity, and Value**.

## ⚖️ 5. Ethical Data Analysis
Every component of the data analysis process must be conducted ethically, considering both moral and legal (GDPR) points of view.
* **Example**: Anonymising customer loyalty data before analysis to ensure privacy.
