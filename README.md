# Data Analysis Portfolio: NCFE Level 2 Certificate in Data

Welcome to my Data Analysis portfolio. This repository serves as a central hub for the coursework, assignments, and practical projects I am completing as part of my **NCFE Level 2 Certificate in Data** with [Netcom Training](https://www.netcomtraining.co.uk/course/certificate-in-data/).

The goal of this repository is to document my journey from data fundamentals to professional analysis, showcasing my ability to handle, process, and visualise data effectively.

## 📌 Course Overview
The NCFE Level 2 Certificate in Data focuses on the essential skills required to work in a data-driven environment. Key areas of study include:
* Understanding the data life cycle.
* Data acquisition and data quality.
* Data analysis tools and techniques.
* Data visualisation and presentation.
* Legal and ethical considerations (GDPR, data security).
